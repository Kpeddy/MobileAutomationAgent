# 📱 Mobile Automation Agent

An AI-driven mobile test automation agent using **MCP (Model Context Protocol) Server** + **Appium (WebDriverIO)** + **BrowserStack App Automate**.

> Full setup instructions coming in the next commit. See the src/ directory for all source files.

## Quick Start

```bash
npm install
cp .env.example .env
npm run upload /path/to/app.apk
npm start
```
